Heat Capacity Unit[i]

  • Default Expression: 800 * 2800 J/m3-C or 0.19107672 * 174.798316 BTU/ft3-F; See EEUNIT
  • Related HST3D Variable: RCPPM
  • HST3D Data Set Record Number: 2.10.1
  • Related HST3D Grid Parameter: Heat Capacity Element Layer[i]
Heat capacity of the porous-medium solid phase per unit volume. The default value is for
800 J/kg-C * 2800 kg/m3
where 800 J/kg-C is an average specific heat of minerals (Fowle, 1923) and 2800 kg/m3 is an average rock density (Clark, 1951).
