Longitudinal

  • Related HST3D Variable: ALPHL
  • HST3D Data Set Record Number: 2.11
  • Related HST3D Grid Parameter: Longitudinal Dispersivity Element Layer[i]
Longitudinal dispersivity

If the longitudinal dispersivity is too low compared to the grid size, you may not get good results.

A conservative value for dispersivity is 1/2 the maximum grid size.

See FDSMTH for more information.
